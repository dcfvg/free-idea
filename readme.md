Free Idea
====

À partir d’un stock de dessins préalablement découpés en milliers de parties, _free idea_ recompose de nouveaux dessins.

**Shortcuts**

`r` -> rm last part
`d` -> draw with dot 

`e` -> eraser
`q` -> add element around the same point

`w` -> start webcam

`s` -> change last part
`b` -> add to black list
