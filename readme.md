Free Idea
====

À partir d’un stock de dessins préalablement découpés en milliers de parties, _free idea_ recompose de nouveaux dessins.

## Shortcuts

**part**

`s` — switch last part

`b` — add to last part to black list 

`r` — remove last part

**mode**

`d` — draw with dot 

`q` — add element around the same start point

**general**

`e` — erase all

`w` — start webcam
